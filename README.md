# Dynamic list

![List](./img/list.gif)
> Uma **dynamic list** utilizando React.

## Como utilizar

```sh
$ git clone https://github.com/TacioAntonio/dynamic-list.git
$ cd dynamic-list
$ npm i
$ npm start
```
